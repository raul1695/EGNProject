The Port Manager module provides the following functionality to the overall system:

1. It has an API that allows Real-time updates on water station usage (E.g: it tells the user which water stations are available for use, and which ones are currently being used)

2. It has an API that keeps track of ship schedules and all of the respective information

The Urls.py contains the API-endpoints as well as further information

The core.py contains the module's functionality implementation.
